# Wind Fantasy 3 한국어 패치

Wind Fantasy 3 한국어 패치 배포 전용 저장소입니다.

이 저장소는 게임 본편이나 원본 게임 리소스를 포함하지 않습니다. 사용자가 보유한 Wind Fantasy 3 설치본을 선택하면, 포함된 패치 도구가 한국어 패치를 적용하고 필요할 때 원본 상태로 복원합니다.

- 배포 파일: `wind3-korean-patch-beta-20260602-v1.zip`

- 파일 크기: `19,043,991` bytes
- 체크섬 파일: `SHA256SUMS.txt`
- 주요 변경사항: [RELEASE_NOTES.md](RELEASE_NOTES.md)
- 버전명 정책: [VERSIONING.md](VERSIONING.md)

## 이번 릴리즈

- 압축 해제 후 실행 파일이 `Wind3 Korean Patch Launcher.exe`로 바뀌었습니다.
- 월드맵과 시나리오의 지명 표기를 같은 기준으로 정리했습니다.
- `성아리온성` / `성아리온평원`은 `성 아리온 성` / `성 아리온 평원`으로, `요시아북광산` / `요시아남광산`은 `요시아 북광산` / `요시아 남광산`으로 정리했습니다.
- `우드삼림`, `웨디진`, `니시므`, `헤미촌`, `샛별성당`, `심수미궁`, `적갈사막`, `장마굴`, `세렌`, `레나`, `조음안`, `설랑애`, `흑우화산` 등 월드맵 라벨을 더 자연스러운 표기로 다듬었습니다.
- 저장/불러오기 화면의 챕터 제목과 전투 제목을 더 자연스럽게 보강했습니다.
- 무기, 방어구, 도구, 스킬, 보석 데이터의 한국어 이름과 설명을 최신 검수본으로 갱신했습니다.
- 패치 적용 시 기존 세이브에 저장된 보유 아이템 이름과 일부 저장 제목도 함께 보정됩니다.
- 이번 배포는 pre-release입니다.

## 사용 방법

1. `wind3-korean-patch-beta-20260602-v1.zip`을 다운로드합니다.
2. 원하는 위치에 압축을 풉니다.
3. 압축을 푼 폴더에서 `Wind3 Korean Patch Launcher.exe`를 실행합니다.
4. 런처에서 `WIND3.EXE`가 있는 Wind Fantasy 3 설치 폴더를 선택합니다.
5. `패치 적용`으로 한국어 패치를 적용합니다.
6. 기존 저장 데이터를 계속 사용할 경우 세이브 패치가 패치 적용 단계에서 자동으로 함께 실행됩니다. 필요하면 런처에서 수동으로 다시 실행할 수도 있습니다.

자세한 런처 사용법과 문제 해결 안내는 압축 파일 안의 `README.txt`를 확인하세요.

## 사용 전 주의사항

- 패치 적용 전에는 게임을 완전히 종료해 주세요.
- 중요한 저장 데이터는 패치 적용 전에 별도로 백업하는 것을 권장합니다.
- 문제가 생기면 런처에서 같은 게임 폴더를 다시 선택한 뒤 `원본 복원`으로 되돌릴 수 있습니다.
- 일부 기존 세이브의 월드맵 라벨은 저장 시점에 따라 이전 표기가 남을 수 있습니다.
- 이 패치는 비공식 개인 한국어 패치이며, 모든 PC 환경에서 동일하게 동작한다고 보장하지 않습니다.
- 일부 이미지형/헤더형 UI 문구는 아직 후속 보강 대상일 수 있습니다.

## 포함하지 않는 것

이 배포 파일은 다음 항목을 포함하지 않습니다.

- Wind Fantasy 3 게임 실행 파일
- 원본 게임 리소스 전체
- 사용자의 저장 데이터
- 로컬 테스트 로그와 개발 중간 산출물
