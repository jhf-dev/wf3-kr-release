# Wind Fantasy 3 한국어 패치

Wind Fantasy 3 한국어 패치 배포 전용 저장소입니다.

이 저장소는 게임 본편이나 원본 게임 리소스를 포함하지 않습니다. 사용자가 보유한 Wind Fantasy 3 설치본을 선택하면, 포함된 패치 도구가 한국어 패치를 적용하고 필요할 때 원본 상태로 복원합니다.

## 이번 릴리즈

- 기본 safe 런타임에서 source-text용 `prepare-load`를 더 이상 실행하지 않도록 해 로드 직전 멈춤을 줄였습니다.
- 캐릭터 정보/스킬 화면 exact-map 적용 뒤 중복 검증 스캔을 릴리즈 기본 경로에서 제거해 로드 후 처리 비용을 줄였습니다.
- 기존 v10의 helper 경로 수정과 라벨 보강은 유지됩니다.
- 배포 환경에서 런타임 helper가 개발용 `analysis_out` 경로를 찾던 문제 수정
- 스킬 화면 `잔여점수`, 캐릭터 정보창 `설명`, `소유물품` 라벨 적용 경로 보강
- 캐릭터 정보창 좌측/중앙 스탯 라벨이 일부 PC에서 원문으로 남던 문제 재보강
- live 검증 결과가 최종 배포 ZIP까지 빠짐없이 들어가도록 exact-byte 검증 강화
- 스킬 화면의 잔여점수/설명 라벨 보강
- live에서 검증된 표면이 릴리즈 산출물에 누락되지 않도록 검증 기준 강화
- 캐릭터 정보창 스탯 라벨 표기 추가 보강
- 캐릭터 정보창 기본/장비 라벨 표기 추가 보강
- 다른 PC 문제 확인을 위한 진단 로그 수집 범위 확대
- 캐릭터 정보창 스탯 표기 복구
- 다른 PC에서 런타임 DLL이 로드되지 않을 가능성을 줄이도록 `d3d9.dll` 의존성 축소
- 원격 환경 문제 확인을 위한 `진단 정보 저장` 기능 추가
- 런타임 패치 기능을 `runtime_policy.ini`에서 제어하도록 개선
- 런타임 영역 패치 적용 방식 조정
- 스토리 번역 품질 1차 폴리싱
- 일부 미번역 UI, 이름, 항목 텍스트 보완
- 챕터 제목 이미지 일부 한글화
- 런처에 게임 설치 경로 감지 기능 추가

## 사용 방법

1. `wind3-korean-patch-beta-20260519-v11.zip`을 다운로드합니다.
2. 원하는 위치에 압축을 풉니다.
3. 압축을 푼 폴더에서 `Wind3 Korean Patch Launcher.vbs`를 실행합니다.
4. 런처에서 `WIND3.EXE`가 있는 Wind Fantasy 3 설치 폴더를 선택합니다.
5. `패치 적용`으로 한국어 패치를 적용합니다.
6. 기존 저장 데이터를 계속 사용할 경우 `세이브 패치` 기능도 함께 실행하는 것을 권장합니다.

자세한 런처 사용법과 문제 해결 안내는 압축 파일 안의 `README.txt`를 확인하세요.

## 사용 전 주의사항

- 패치 적용 전에는 게임을 완전히 종료해 주세요.
- 중요한 저장 데이터는 패치 적용 전에 별도로 백업하는 것을 권장합니다.
- 문제가 생기면 런처에서 같은 게임 폴더를 다시 선택한 뒤 `원본 복원`으로 되돌릴 수 있습니다.
- 이 패치는 비공식 개인 한국어 패치이며, 모든 PC 환경에서 동일하게 동작한다고 보장하지 않습니다.
- 일부 이미지형/헤더형 UI 문구는 아직 후속 보강 대상일 수 있습니다.

## 포함하지 않는 것

이 배포 파일은 다음 항목을 포함하지 않습니다.

- Wind Fantasy 3 게임 실행 파일
- 원본 게임 리소스 전체
- 사용자의 저장 데이터
- 로컬 테스트 로그와 개발 중간 산출물
