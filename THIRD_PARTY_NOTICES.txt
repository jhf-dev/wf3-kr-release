# 서드파티 구성요소 및 배포 정책

이 파일은 공개 최종 사용자 패키지에 실제 포함되거나, 패키지의 생성 자산에 사용된 서드파티 구성요소를 고지합니다.

이 배포물은 정식 게임 설치본에 적용하는 비공식 한국어 패치입니다. W3BP2 차이 데이터만 배포하며 `WIND3.EXE`, 완전한 원본 게임 파일, 완전한 패치 게임 파일을 포함하지 않습니다.

프로젝트가 작성한 런처·패치 코드, 번역문, 패치 데이터에는 별도 허가가 없는 한 오픈소스 라이선스가 부여되지 않습니다. 아래 서드파티 라이선스는 해당 서드파티 구성요소에만 적용됩니다.

## 공개 패키지의 서드파티 구성요소

### CPython 3.11.8
- 라이선스: Python Software Foundation License
- Upstream: https://www.python.org/downloads/release/python-3118/
- 전문: `licenses/CPython-3.11.8-LICENSE.txt`

### PyInstaller 6.20.0
- 라이선스: GNU GPL v2 or later with the PyInstaller Bootloader Exception; the full upstream terms also identify separately licensed runtime hooks
- Upstream: https://pyinstaller.org/
- 전문: `licenses/PyInstaller-6.20.0-COPYING.txt`

### OpenSSL 3.0.13
- 라이선스: Apache License 2.0
- Upstream: https://www.openssl.org/
- 전문: `licenses/OpenSSL-3.0.13-LICENSE.txt`

### libffi
- 라이선스: MIT-style license
- Upstream: https://github.com/libffi/libffi
- 전문: `licenses/libffi-LICENSE.txt`

### Microsoft Visual C++ Runtime
- 라이선스: Microsoft redistributable terms (not open source)
- Upstream: https://learn.microsoft.com/cpp/windows/latest-supported-vc-redist
- 전문: `licenses/Microsoft-VC-Runtime-NOTICE.txt`

### Pretendard
- 라이선스: SIL Open Font License 1.1
- Upstream: https://github.com/orioncactus/pretendard
- 전문: `licenses/Pretendard-OFL-1.1-LICENSE.txt`

Pretendard 폰트 파일 자체는 공개 패키지에 포함하지 않으며, 고정된 upstream 폰트에서 생성한 게임용 글리프 패치만 포함할 수 있습니다.

개발용 XLSX 런처는 공개 배포 대상이 아니며, 이 고지의 적용 범위와 공개 최종 사용자 ZIP 구성요소에 포함되지 않습니다.

게임 원본과 그 파생 자산에 대한 권리는 원저작권자에게 있으며, 이 고지는 그 권리를 변경하거나 재배포 권한을 부여하지 않습니다.
