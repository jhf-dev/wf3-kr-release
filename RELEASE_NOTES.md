# 릴리즈 노트 (beta-20260519-v7)

## 다운로드

- 배포 파일: `wind3-korean-patch-beta-20260519-v7.zip`
- SHA-256: `EBDB7280A45967931C44465531AB8099B310DC6B18437470A877254BA2B8E2DC`
- 파일 크기: `18,931,485` bytes

## 주요 변경사항

* 캐릭터 정보창 스탯 라벨 추가 보강
  - 다른 PC에서 캐릭터 정보창 스탯 라벨이 원문으로 남던 문제를 추가 보강했습니다.
  - 스탯 라벨 문자열의 실제 게임 출력 형식에 맞춰 런타임 owner rewrite 매칭을 다시 조정했습니다.

* 릴리즈 전 산출물 검증 강화
  - 런타임에서 확인된 수정이 최종 배포 ZIP까지 들어갔는지 별도로 검증하도록 기준을 강화했습니다.
  - 패키지 안에 필요한 런타임 적용 경로와 DLL 내부 표식이 빠지면 릴리즈 빌드가 실패하도록 보강했습니다.

* 캐릭터 정보창 기본/장비 라벨 추가 보강
  - 다른 PC에서 캐릭터 정보창의 `레벨`, `직업레벨`, `생명`, `정신`, `투지`, 장비 슬롯 라벨 일부가 원문으로 남던 문제를 보강했습니다.
  - 런타임 owner rewrite가 실제 게임 문자열의 콜론 포함 형식과 맞도록 수정했습니다.

* 진단 로그 수집 범위 확대
  - `진단 정보 저장` 결과에 startup bridge 내부 로그, 런타임 패치 cycle report, worker stdout/stderr 로그를 추가했습니다.
  - 다른 PC에서만 실패하는 경우에도 어느 적용 단계에서 실패했는지 더 빨리 확인할 수 있습니다.

* 다른 PC에서의 런타임 DLL 로드 안정성 보강
  - 패치용 `d3d9.dll`의 외부 C 런타임 의존성을 제거해, PC 환경 차이로 DLL이 로드되지 않을 가능성을 줄였습니다.
  - 기존 기능인 캐릭터 정보창 스탯 표기 복구와 런타임 기능 설정은 유지됩니다.

* 원격 환경 진단 기능 추가
  - 런처에 `진단 정보 저장` 기능을 추가했습니다.
  - 다른 PC에서만 문제가 발생할 때 선택한 게임 폴더, 주요 파일 해시, 런타임 설정, 관련 로그, 최근 Windows 오류 기록을 한 번에 저장할 수 있습니다.

* 캐릭터 정보창 스탯 표기 복구
  - 일부 환경에서 캐릭터 정보창의 스탯 라벨이 원문으로 되돌아가던 문제를 수정했습니다.
  - 엔진 레벨의 owner trace/rewrite 적용 경로가 릴리즈 DLL에 포함되도록 바로잡았습니다.

* 런타임 기능 설정 방식 개선
  - `runtime_policy.ini`의 `[runtime_features]` 섹션에서 owner trace/rewrite, 모듈 UI 패치, 진단용 프로브 같은 런타임 기능을 제어할 수 있게 했습니다.
  - 같은 종류의 기능 전환을 위해 매번 DLL을 새로 빌드해야 하는 부담을 줄였습니다.

* 릴리즈 패키징 검증 강화
  - 릴리즈 빌더가 Phase 1C owner-rewrite용 런타임 DLL인지 확인한 뒤 패키징하도록 검증을 추가했습니다.
  - 잘못된 DLL이 다시 배포 ZIP에 섞이는 상황을 조기에 차단합니다.

* 런타임 영역 패치 적용 방식 조정
  - 런처 실행 직후와 로드 이후 UI 패치가 더 빠르게 반영되도록 적용 타이밍을 보강했습니다.

* 스토리 번역 품질 1차 폴리싱
  - 주요 대사의 표현, 말투, 용어를 1차로 정리해 읽기 흐름을 개선했습니다.

* 일부 미번역 대응
  - 플레이 중 노출되던 UI, 이름, 항목 텍스트 일부에 한국어 대응을 추가했습니다.

* 챕터 제목 이미지 한글화
  - 이미지로 표시되는 챕터 제목 일부를 한국어로 교체했습니다.

* 런처 게임 경로 감지 기능 추가
  - Steam 설치 경로에 있는 `Wind Fantasy 3` 폴더를 자동으로 감지하는 기능을 추가했습니다. 폴더명이 다른 경우 수동 선택이 필요할 수 있습니다.

## 사용 전 주의사항

- 이 배포 파일은 게임 본편이나 원본 게임 리소스를 포함하지 않습니다. 사용자가 보유한 Wind Fantasy 3 설치본이 필요합니다.
- 스팀 클라이언트 기준 2025-08-12 버전을 기반으로 개발했습니다. 이외 버전에서는 패치 적용 및 동작을 보장하지 않습니다.
- 패치 적용 전에는 게임을 완전히 종료해 주세요. 게임이 실행 중이면 일부 파일 교체가 실패하거나 적용 결과가 일관되지 않을 수 있습니다.
- 중요한 저장 데이터는 패치 적용 전에 별도로 백업하는 것을 권장합니다.
- 이 패치는 비공식 개인 한국어 패치입니다. 모든 PC 환경에서 동일하게 동작한다고 보장하지 않습니다.
- 일부 이미지형/헤더형 UI 문구는 아직 후속 보강 대상일 수 있습니다.
